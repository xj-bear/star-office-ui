## 中文说明 (Chinese)

这是一个为 OpenClaw 开发的**像素风云端办公室**。它可以根据 OpenClaw 的实时工作状态，让你的像素小人同步进行“写代码”、“查资料”、“摸鱼”等动作，并带有高度自定义的管理后台。

### ✨ 特性
- **实时状态同步**：通过监听 OpenClaw 日志自动同步 Agent 的工作状态。
- **12+ 像素 IP 皮肤**：内置史莱姆、小幽灵、猫咪、兔子、熊猫等 12 种极其可爱的动态皮肤。
- **可视化管理后台**：支持在线上传背景、选择皮肤、在地图上拾取定位坐标、配置随机话术。
- **趣味互动**：点击画布任何位置，小人会自动走过去并触发吐槽，内置“茶水间”特殊导航逻辑。

### 🚀 快速启动
1. **安装依赖**: `pip install -r requirements.txt`
2. **生成资源**: 执行所有的 `generate_*.py` 脚本以生成像素图集。
3. **启动**: 执行 `./start.sh`。访问 `http://localhost:18888`。

## English Version

A **Pixel-Art Cloud Office** built for OpenClaw. It synchronizes your pixel avatar's movements and behaviors (like "coding", "researching", or "slacking") based on OpenClaw's real-time activities.

### ✨ Features
- **Real-time Sync**: Automatically tracks Agent status via OpenClaw logs.
- **12+ Pixel IPs**: Includes 12+ adorable animated skins (Slime, Ghost, Cat, Bunny, Panda, etc.).
- **Admin Dashboard**: Web-based UI to upload backgrounds, pick skins, define coordinates, and customize dialogue.
- **Interactive**: Characters react to clicks, navigate to the cursor, and have special logic for the "Coffee Break" area.

### � Quick Start
1. **Dependencies**: `pip install -r requirements.txt`
2. **Assets**: Run all `generate_*.py` scripts to build the sprite sheets.
3. **Run**: Execute `./start.sh`. Visit `http://localhost:18888`.
